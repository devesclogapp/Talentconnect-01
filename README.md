# Execute e implemente seu app do AI Studio

Este repositório contém tudo o que você precisa para rodar seu aplicativo localmente.

Visualize seu app no AI Studio: https://ai.studio/apps/drive/1FR4sFJ_p5RbJ1e1EMM6T-ngOS1uxF5Kq

## Executando Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure a `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com sua chave da API Gemini.
3. Inicie o aplicativo:
   `npm run dev`
